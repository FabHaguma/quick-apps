# Quick Apps

A collection of useful lightweight JavaScript utility applications.

**Live Demo:** [https://quickapps.haguma.com/](https://quickapps.haguma.com/)

## Included Applications

*   **BMI Calculator**: Calculate Body Mass Index based on height and weight.
*   **Fuel Consumption Converter**: Convert fuel efficiency (e.g., MPG to km/L).
*   **Prime Numbers**: Tools related to prime number calculation.
*   **Currency Converter**: Real-time currency conversion rates.
*   **Alcohol Tracker**: Track and calculate units or BAC.
*   **Job File Namer**: A utility to generate consistent file names for job applications.

## How to Run Locally

1.  Clone the repository.
2.  Open `index.html` in your web browser.
3.  Navigate to the specific app you want to use.
